# webBoilerplate
Development Environment and boilerplate for Web page.

## Features

- webpack
- Babel
    * ES2015
- Sass
    * precss
    * Autoprefixer
        * last 2 versions', '> 2%'
    * postcss-media-minmax
    * cssnano

## Usage

```
$ yarn install

$ yarn run watch
```
